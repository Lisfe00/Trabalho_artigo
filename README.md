# Sistema de artigos

* Feito com Node.js e Express.js
* Encode: UTF-8
* http://172.17.0.1:8069

## NPM

Instalar pré-requisitos:

* [Node 18.17.1 LTS](https://nodejs.org/en)

Instale as dependencias

```sh
$ npm install
```

Execute o node

```sh
$ npm start
```

Arquivos de estilização devem estar na pasta /public
